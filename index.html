From copy to shipped

Never pulled a script into Studio before? It takes about a minute. Here's the whole flow.

1

Pick a script

Browse the library or a game section and find the system you need.

2

Copy the code

Hit the copy icon on any card — the full Luau source lands on your clipboard instantly.

3

Paste into Studio

Drop it into a Script, LocalScript, or ModuleScript depending on what it's built for.

4

Test & tweak

Run it in Studio's playtest mode, adjust the values to fit your game, then publish.

How to make this yours

Everything on this page — library, filters, and all three game sections — renders from one JavaScript array. No backend, no database.

01

Find the SCRIPTS array

Open the page source and look for const SCRIPTS = [...] near the end of the file.

02

Tag it with a game

Set game to "gag2", "mm2", "adoptme", or "general" and it's routed automatically.

03

Save and reload

Grids, counts, search, filters, and syntax highlighting all rebuild themselves from the array.

04

Get notified of visits

Find DISCORD_WEBHOOK_URL near the top of the script and paste in a webhook link. The message updates itself as visitors browse — showing time on site and which scripts they copied — right up until they leave.
